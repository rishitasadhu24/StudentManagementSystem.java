# Student Management System (Java)

##  Project Description
This is a simple menu-driven Student Management System developed using Java.
The project demonstrates Object-Oriented Programming concepts like Inheritance
and basic CRUD operations using arrays.


## Features
- Add Student
- View All Students
- Search Student by Roll Number
- Delete Student
- Menu-driven console interface


## Concepts Used
- Classes and Objects
- Inheritance
- Arrays
- Loops
- Conditional Statements
- Scanner Class
- Basic CRUD Operations


##  Technologies Used
- Java
- VS Code / IntelliJ
- Git & GitHub



##  How to Run the Project

1. Open terminal inside project folder.
2. Compile the program:

   javac StudentManagementSystem.java

3. Run the program:

   java StudentManagementSystem
